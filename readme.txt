== Description ==
Any Web form integration into wordpress website. It can be Zoho webform, Infusionsoft webform, sugar crm, suite crm or any webform. Shortcode option to place form at any place in website. 

Use shortcode **[webform]**

<a href="http://www.miraclewebsoft.com/webform-integration/">Check Video Tutorial</a>

**How to use plugin:-** 

* Install plugin.
* Now goto webform-integration under Settings in admin menu.
* Place given code into Form code then save.
* Place **[webform]** any where in website.

**Plugin features:-** 

* Zoho webform integration
* Infusionsoft webform integration
* Suitecrm/sugar webform integration
* Any webform
* Shortcode option to place form any where in website


== Installation ==
Download the plugin .zip file

Login in to admin Click Plugins -> Add New -> Upload

Find webform-integration Wordpress Plugin in plugin list and activate it.

Now goto webform-integration under Settings in admin menu.

Place given code into Form code then save.

Place **[webform]** any where in website.
